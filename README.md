# Azure-Cloud-Administration-Project
## Overview
This project demonstrates the implementation of a Microsoft Azure cloud envionment for CTU Training Solution. The project covers networking, virtual machines, security, RBAC, monitoring and cost management.
The practical activities were completed over a 9-week period as part of my module NET731. 

---

## Technologies Used

- Microsoft Azure
- Microsoft Entra ID
- Azure Virtual Machines
- Azure Monitor
- Microsoft Defender for Cloud
- Azure Storage Accounts
- Network Security Groups
- Azure Cost Management

---

## Project Objectives

- Design and deploy an Azure cloud environment
- Configure secure networking and access controls
- Implement identity and role management
- Monitor cloud resources and performance
- Apply security recommendations
- Manage cloud costs effectively
